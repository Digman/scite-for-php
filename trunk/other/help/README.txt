编译：

  $ cd source/scintilla/win32
  $ mingw32-make
  $ cd ../..
  $ cd scite/win32
  $ mingw32-make

安装：

  把 source/scite/bin/scilexer.dll 和 source/scite/bin/scite.exe 复制到根目录即可。

高效率文本编辑原则：

  1、寻找繁复的、无聊的操作；
  2、简化、自动化该操作；
  3、让这种简化和自动化成为习惯。

快捷键：

  Ctrl + ? : 最常用
  Alt  + ? : 次常用
  Ctrl + Shift + ? : 一般使用
  ... ...

特性：

  能做到的（有些条目不确定，或能做到但暂未实现）：
  
    * 定制编辑器颜色

        各种编辑器元素的样式都应该能定制

    * 词法高亮

        能识别各种语言，能定制各种 token，以分别设置样式

    * 自定义快捷键

        编辑器的各个功能都可以绑定到某个快捷键，而且某个功能可以绑定多个快捷键

    * snippet

        类似 textmate 的 snippet

    * 功能宏

        把编辑器的某一连串命令封装成一个宏，绑定到某个快捷键

    * 列编辑

        @todo

    * 自动探测文本编码 [待实现]

        至少能识别 gbk 和 utf-8

    * 一起缩进一起回退

        选中某几行，按 Tab 即一起缩进一个单元，按 Shift+Tab 则回退一个缩进单元

    * 代码折叠

        

    * 自动补全(单词、括号、单双引号)
    * 字符串缩写
    * API 提示
    * 查找与替换

        支持在多 buffer 内查找和替换，在文件系统里查找，支持正则表达式

    * ... ...
    * ... ...
    * ... ...

  做不到的（有些条目不确定，或能做到但暂未实现）：
  
    * 分屏（单个 tab/buffer 分成 screen，多个 tab/buffer 分成 window）
    * 文件浏览器
    * 在文件系统里替换
